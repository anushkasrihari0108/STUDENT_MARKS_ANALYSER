# Student Marks Analyser

## About the Project
This is a simple Python project where I used dictionary and list comprehensions to analyze student marks.
The goal was to practice writing clean and efficient code while performing basic data analysis.

## What This Project Does
* Checks whether each student **passed or failed**
* Assigns **grades (A, B, C, F)** based on marks
* Creates a list of **passed students**
* Creates a list of **failed students**
* Identifies **top-performing students**

## Technologies Used
* Python
* Dictionary Comprehension
* List Comprehension

## Example Output

You will see output like:

```id="cmd002"
RESULT: {'SMITH': 'Pass', 'SCOTT': 'Pass', 'MARTIN': 'Fail', ...}
GRADES: {'SMITH': 'A', 'SCOTT': 'B', 'MARTIN': 'F', ...}
PASSED STUDENTS: ['SMITH', 'SCOTT', 'ALLEN', 'FORD', 'VIRAT']
FAILED STUDENTS: ['MARTIN', 'MILLER']
TOPPERS: ['SMITH', 'ALLEN', 'VIRAT']
```
